
# Reading Companion Frontend

This directory contains the Reading Companion web client.

It started from a Figma Make export, but it is now integrated with the backend API contract documented at the workspace root.

## Source Of Truth
- Workspace overview: `../README.md`
- API integration: `../docs/api-integration.md`

## Running Locally
- From the workspace root, prefer:
  - `make dev-frontend`
  - `make dev`
- From this directory only:
  - `npm install`
  - `npm run dev`
  - `npm run build`
